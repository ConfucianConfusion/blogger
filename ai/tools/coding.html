Github Copilot之外辅助编程的工具，让你事半功倍

发布于：2025-03-09

毫无疑问，时至今日，github copilot依然是最为主流的ai辅助编程工具，琳琅满目的Claude 3.7 Sonnet，GPT-o3mini，Gemini 2.0等等 尤其是拿到学生认证后的免费使用，让穷苦的学生党切身体会到了白嫖的快感。但是，github copilot对申请者学校有着严格限制，好家伙学历歧视是吧， 这对大多数人来说无疑提出了挑战（不过也可以用一些手段骗过去），所以，有没有其他的ai辅助编程工具呢？ 有的兄弟，有的。

*Github Copilot支持的模型，上百刀的服务免费用是真的香((o(*>ω<*)o))

1. Cursor

Anysphere工作室基于vscode框架以及主流一些ai模型所开发的工具，感觉是现在热度很高的一款产品（？

直接到官网上下载就好了，使用教程知乎这篇文章有写道：https://zhuanlan.zhihu.com/p/23874722853

目前有Claude 3.7 Sonnet，GPT-4.5之类的模型可供使用，登录后会送一些免费次数，之后就要氪金了

2.Trae

字节跳动基于vscode框架以及主流一些ai模型所开发的工具，应该是国内这一块做的比较好的产品

直接到官网上下载就好了，目前支持的模型比较少，只有Deepseek-v3和Claude 3.5 Sonnet

好消息是免费使用，不过一天似乎有限流。这个网站的样式设计与交互功能很多就是在这个IDE上完成的。

3.亚马逊推出的插件

这一款的好处也是个人用免费，我还没用过，不过市面上没有多大的反响，可能编程能力稍微逊色一点？

各个IDE都上架了它的插件，我这里展示vscode安装界面
